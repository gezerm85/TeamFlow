## 📖 Hakkında

**TeamFlow**, modern web teknolojileri kullanılarak geliştirilmiş kapsamlı bir takım yönetimi ve organizasyon şeması uygulamasıdır. React, TypeScript ve ReactFlow ile oluşturulmuş olup, kullanıcılara görsel takım organizasyonu, interaktif diyagramlar ve detaylı analitik raporlar sunmaktadır.

### 🎯 Projenin Amacı

- 👥 **Takım Yönetimi** - Kolay takım oluşturma ve düzenleme
- 📊 **Organizasyon Şeması** - Görsel takım hiyerarşisi
- 📈 **Analitik Raporlar** - Detaylı takım istatistikleri
- 🎨 **Modern UI/UX** - Kullanıcı dostu arayüz
- 📱 **Responsive Design** - Tüm cihazlarda uyumlu
- 🔄 **Real-time Updates** - Anlık veri güncellemeleri

## 🚀 Kurulum

### Gereksinimler

- **Node.js** (v18 veya üzeri)
- **npm** veya **yarn**
- **TypeScript** (v5.7.2 veya üzeri)

### Adım Adım Kurulum

1. **Depoyu klonlayın**
   ```bash
   git clone https://github.com/gezerm85/TeamFlow.git
   cd TeamFlow
   ```

2. **Bağımlılıkları yükleyin**
   ```bash
   npm install
   # veya
   yarn install
   ```

3. **Uygulamayı başlatın**
   ```bash
   # Geliştirme sunucusunu başlat
   npm run dev
   
   # Build oluştur
   npm run build
   
   # Preview
   npm run preview
   ```

### Build Komutları

```bash
# Development
npm run dev

# Build
npm run build

# Lint
npm run lint

# Preview
npm run preview
```

### Context API Yapısı

```typescript
interface Team {
  id: string;
  name: string;
  users: User[];
}

interface User {
  id: string;
  name: string;
}

interface TeamContextProps {
  teams: Team[];
  addTeam: (teamName: string) => void;
  addUserToTeam: (teamId: string, userName: string) => void;
  removeUserFromTeam: (teamId: string, userId: string) => void;
  updateTeam: (teamId: string, newName: string) => void;
  removeTeam: (teamId: string) => void;
  updateUser: (teamId: string, userId: string, newName: string) => void;
}
```

### ReactFlow Kullanımı

```typescript
import ReactFlow, { Controls, Background, Node, Edge } from 'reactflow';

const nodes: Node[] = [
  {
    id: '1',
    type: 'input',

const edges: Edge[] = [
  { id: 'e1-2', source: '1', target: '2' },
```
